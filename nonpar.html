Nonparametric two-sample testing

Kemeng Zhang

2018-04-20

Introduction

The nonparametric two-sample hypothesis testing problem involves: given X_1, X_2, …, X_n i.i.d. F and Y_1, Y_2, …, Y_m i.i.d. G, test the null hypothesis of F = G against the alternative hypothesis of F is not equal to G. The test statistic is based on embedding F and G into a reproducing kernel Hilbert space and then compute a distance between the resulting embeddings. For this primitive, the Hilbert space is associated with the Gaussian kernel.

Testing/Simulation

Stochastic Blockmodel Example

We illustrate the hypothesis tests through several simulated and real data examples. For our first example, let F_epsilon for a given epsilon > 0 be mixture of point masses corresponding to a two-block stochastic block model with block membership probabilities (0.4,0.6) and block probabilities 0.5+epsilon if in the same block, 0.2 otherwise. We then test, for a given epsilon > 0, the hypothesis whether F and F_epsilon are the same.

# Generate Adjacency Matrix
n <- 100
blocks <- c(rep(0, n*0.4), rep(1, n*0.6))
A <- matrix(rep(0, n*n), nrow = n)
for (i in 2:n) {
  for (j in 1:(i-1)) {
    if (blocks[i] == blocks[j]) {
      A[i, j] <- rbinom(1, 1, 0.5)
    } else {
      A[i, j] <- rbinom(1, 1, 0.2)
    }
    A[j, i] <- A[i, j]
  }
}
g <- igraph::graph_from_adjacency_matrix(A)
plot(g)

# Generate Adjacency Matrix with epsilon = 0.02
n <- 100
epsilon <- 0.02
blocks <- c(rep(0, n*0.4), rep(1, n*0.6))
Ahat <- matrix(rep(0, n*n), nrow = n)
for (i in 2:n) {
  for (j in 1:(i-1)) {
    if (blocks[i] == blocks[j]) {
      Ahat[i, j] <- rbinom(1, 1, 0.5 + epsilon)
    } else {
      Ahat[i, j] <- rbinom(1, 1, 0.2)
    }
    Ahat[j, i] <- Ahat[i, j]
  }
}
ghat <- igraph::graph_from_adjacency_matrix(A)
plot(ghat)

# Embed to R^2
dim = 2
Xhat = graphstats::ase(A,dim)
X_epsilon_hat = graphstats::ase(Ahat,dim)
test_stats_1 = graphstats::nonpar(Xhat,X_epsilon_hat,sigma = 0.5)
print(test_stats_1)
## [1] 231.4845

We test if the estimated block assignment is approximately the same as the one we intended.

First, test Graph G.

# Cluster using EM algorithm.
model <- Mclust(Xhat)
cat("The estimate number of classes (blocks) is:", model$G, "\n")
## The estimate number of classes (blocks) is: 2
predictions <- round(model$z[,2])

# Calculate accuracy.
rate <- sum(as.numeric(predictions == blocks))/n
cat("The classification accuracy is:", rate, "\n")
## The classification accuracy is: 1

Second, test Graph G_epsilon.

# Cluster using EM algorithm.
model <- Mclust(X_epsilon_hat)
cat("The estimate number of classes (blocks) is:", model$G, "\n")
## The estimate number of classes (blocks) is: 2
predictions <- round(model$z[,2])

# Calculate accuracy.
rate <- sum(as.numeric(predictions == blocks))/n
cat("The classification accuracy is:", rate, "\n")
